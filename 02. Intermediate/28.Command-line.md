
# Command Line Arguments

- We can pass some parameters or flags to our program when executed from a terminal or command line.
- To access those passed values in the program, we use command line arguments.
- We can read the command line arguments using `os.Args` method of `os` package.


## os.args

- Returns the string slice of al the parameters passed from the command line or terminal.
- First argument returns the complete path of the executable that was generated.
