
# Command Line Arguments

- We can pass some parameters or flags to our program when executed from a terminal or command line.
- To access those passed values in the program, we use command line arguments.
- We can read the command line arguments using `os.Args` method of `os` package.


## os.args

- Returns the string slice of al the parameters passed from the command line or terminal.
- First argument returns the complete path of the executable that was generated. First command will always to run the source code.

```go
func main() {
	for index, arg := range os.Args {
		fmt.Println("ARGUMENT::", index, " ", arg)
	}
}
ARGUMENT:: 0   /var/folders/qq/jx88nr752bv919fssgg4yjy00000gn/T/go-build2446194451/b001/exe/hello-world
ARGUMENT:: 1   saksham
ARGUMENT:: 2   gupta
ARGUMENT:: 3   hello
```

## Flags

- Values we provide with `-` or `--`. Like `--help`.
- Read the value of a flag with this name and store it in this variable.
- It takes 3 inputs, address of the variable, name of the flag to read, default value in case user did not provide any value and a description of that flag.

```go
var name string
var age int
var single bool

flag.StringVar(&name, "name", "John Doe", "Name of the user")
flag.IntVar(&age, "age", 18, "Age of the user")
flag.BoolVar(&single, "single", true, "Marrital status of the user")

flag.Parse()

fmt.Println("NAME::", name)
fmt.Println("AGE::", age)
fmt.Println("SINGLE::", single)

/* go run hello-world.go -name "saksham gupta" -age 22 -single true

NAME:: saksham gupta
AGE:: 22
SINGLE:: true

*/
```
