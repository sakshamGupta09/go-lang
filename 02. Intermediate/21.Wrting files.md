# Writing files

- I want to create a new file in my OS with some content in it.
- We will be using `os` package to do file opertions.


## os.Create(fileName) *os.File, err

- Creates a new file and returns the instance.
- If the file alraedy exists, its content is replaced with newer one.

```go
func main() {
	file, err := os.Create("logs.text")

	if err != nil {
		fmt.Println("Error creating a file", err)
		return
	}
}
```

## Writing data

- We have to methods on file instance. `Write` and `WriteString`.
- `Write` writes a bye slice whereas `WriteString` writes a string to the file.
- It returns number of bytes written to the file and an error.

```go
func main() {
	file, err := os.Create("logs.text")

	if err != nil {
		fmt.Println("Error creating a file", err)
		return
	}

	file.Write([]byte("Hello world!"))
	file.WriteString("This is a beautiful day")

	defer file.Close()
}
```
