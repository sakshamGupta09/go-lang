# Fmt package

- **Printing** → output to terminal or strings
- **Formatting** → control how values appear
- **Scanning** → read input from user or other sources


## Printing functions

## Print

- No spaces added if we print multiple values.
- No newline at the end. Cursor stays at same line.
- Concatenates values.

```go
func main() {
	fmt.Print("hello", "world", 5) // helloworld5 10%        
}
```

## Println

- Prints the values.
- Adds spaces between them for better readability.
- Takes teh cursor to new line.
