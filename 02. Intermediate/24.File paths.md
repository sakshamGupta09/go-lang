# File paths

- Different OS use different path formats. `filepath` ensures paths are built and parsed correctly for the current OS.
- While reading and writing files we may need to construct a path first to be able to reach that target location.


## filepath.Join()

- Different OS use different seperators to construct a path.
- This method is OS aware and will use teh right seperator. We just need to supply elements to it.

```go
joinedPath := filepath.Join("Desktop", "Projects", "readme.md")

fmt.Println(joinedPath) // Desktop/Projects/readme.md
```

## filepath.clean()

- It removes redundant components from the path.

```go
func main() {
	path := filepath.Clean("./data/../data/file.txt")

	fmt.Println(path) // data/file.txt
}
```

## filepath.split

- Usually a path has directories and then the file name.
- split method returns us directory path and file name.
- The file will always return the last element in path even if thats not a file.

```go
func main() {
	path := filepath.Clean("home/users/projects/readme.md")

	dir, file := filepath.Split(path)

	fmt.Println(dir)  // home/users/projects/
	fmt.Println(file) // readme.md
}
```

## filepath.isAbs

- Returns a boolean telling if a path is absolute or relative.

```go
func main() {
	path := "/home/users/projects/readme.md"
	fmt.Println(filepath.IsAbs(path)) // true
}
```

## filepath.Ext()

- Returns the extension of the file.

```go
func main() {
	path := "/home/users/projects/readme.md"
	fmt.Println(filepath.Ext(path)) // .md
}
```
