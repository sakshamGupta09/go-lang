# Bufio Package

## Buffering

- I want to read a 10GB file located on my computer through my GO program. 
- To do so, I need to make repeated OS calls. OS calls are slow compared to normal memory access.
- Buffering = collecting data in advance and keeping it in main memory (buffer) so we donâ€™t have to fetch it again and again from OS.


## How to create a bufio.Reader

```go
reader := bufio.NewReader(source)
```
- source can be File, os.Stdin, Network connection, Any io.Reader
