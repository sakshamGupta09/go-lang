# Sorting

- We want to sort a collection of items.
- In place sorting happens, original slice is modified.


## Sort integers

- In place sorting of slice.
- No new slice is returned. Original slice is modified.


```go
func main() {
	nums := []int{1, 5, 3, 2, 4}

	sort.Ints(nums)
	fmt.Println(nums)
}
```

## String slice

```go
func main() {
	fruits := []string{"grapes", "orange", "apple", "banana"}

	sort.Strings(fruits)
	fmt.Println(fruits)
}
```
