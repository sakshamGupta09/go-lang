# Sorting

- Always a collection needs to be sorted.


## Slice of numbers

- In place sorting of slice.
- No new slice is returned. Original slice is modified.


```go
func main() {
	nums := []int{1, 5, 3, 2, 4}

	sort.Ints(nums)
	fmt.Println(nums)
}
```

## String slice

```go
func main() {
	fruits := []string{"grapes", "orange", "apple", "banana"}

	sort.Strings(fruits)
	fmt.Println(fruits)
}
```

## Custom sort

- Define custom rules for sorting the data.
- We need to implement `sort.Interface` which consists of three methods
- `Len() int` Returns the size of the collection.
- `Less(i, j int) bool` Compares elements at indices i and j and returns true if element at i should be placed before element at j.
- `Swap(i,j int)` Swaps teh elements at indices i and j.

```go
type Person struct {
	Name string
	Age  int
}

type ByAge []Person

func (b ByAge) Len() int {
	return len(b)
}

func (b ByAge) Less(i, j int) bool {
	return b[i].Age < b[j].Age
}

func (b ByAge) Swap(i, j int) {
	b[i], b[j] = b[j], b[i]
}

func main() {
	people := []Person{
		{Name: "Candy", Age: 50},
		{Name: "Bob", Age: 20},
		{Name: "Alice", Age: 40},
		{Name: "David", Age: 60},
		{Name: "John", Age: 30},
	}

	sort.Sort(ByAge(people))
	fmt.Println(people)
}

```
