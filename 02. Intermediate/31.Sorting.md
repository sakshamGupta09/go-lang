# Sorting

- We want to sort a collection of items.
- In place sorting happens, original slice is modified.


## Sort integers

```go
func main() {
	nums := []int{1, 5, 3, 2, 4}

	sort.Ints(nums)
	fmt.Println(nums)
}
```

## Sort Strings

```go
func main() {
	fruits := []string{"grapes", "orange", "apple", "banana"}

	sort.Strings(fruits)
	fmt.Println(fruits)
}
```

## Sort Floats

```go
floats := []float64{3.2, 1.5, 2.8}
sort.Float64s(floats)
```
