# XML

- XML is another data format used by machines to communicate and exchange data.
- We need to convert go data structures to XML and XML to Go data structures.

## Convert a struct to XML

```go
type Person struct {
	XMLName   xml.Name `xml:"person"`
	FirstName string   `xml:"first_name"`
	LastName  string   `xml:"last_name"`
	Age       int      `xml:"age"`
}

func main() {
	person := Person{FirstName: "John", LastName: "Doe", Age: 26}

	xmlData, err := xml.Marshal(person)

	if err != nil {
		log.Fatal("ERROR", err)
	}

	fmt.Println(string(xmlData))
}
// <person><first_name>John</first_name><last_name>Doe</last_name><age>26</age></person>
```

## Indenting output

- With Marshal, we got the oputput but thats now how XML are typically represented.
- So, we have another method `MarshalIndent`.

```go
type Person struct {
	XMLName   xml.Name `xml:"person"`
	FirstName string   `xml:"first_name"`
	LastName  string   `xml:"last_name"`
	Age       int      `xml:"age"`
}

func main() {
	person := Person{FirstName: "John", LastName: "Doe", Age: 26}

	xmlData, err := xml.MarshalIndent(person, "", "  ")

	if err != nil {
		log.Fatal("ERROR", err)
	}

	fmt.Println(string(xmlData))

	/*
		  <person>
			  <first_name>John</first_name>
			  <last_name>Doe</last_name>
			  <age>26</age>
			</person>
	*/
}
```
> If we omit `XMLName` it will pick the name of the struct.
