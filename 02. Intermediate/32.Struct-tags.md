# Struct Tags

- How `struct` fields should be mapped to a `json` and vice versa.
- To omit certain fields from the `json` output when they have zero value(omitempty) or always (-).

```go
type Person struct {
	FirstName string `json:"first_name"`
	LastName  string `json:"last_name,omitempty"`
	Age       int    `json:"age"`
}

func main() {
	person := Person{FirstName: "John", LastName: "", Age: 26}

	jsonData, err := json.Marshal(person)

	if err != nil {
		log.Fatal("ERROR", err)
	}

	fmt.Println(string(jsonData)) // No lastname {"first_name":"John","age":26}
}
```

## Completely exclude a field

```go
type Person struct {
	FirstName string `json:"first_name"`
	LastName  string `json:"-"`
	Age       int    `json:"age"`
}
```
