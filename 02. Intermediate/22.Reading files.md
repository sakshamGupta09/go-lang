# Reading files

- I want to read the csv file present in my OS.
- When we are done reading the file, there is a EOF character which indicates there is no more data.


## The Simple Way (Read Everything At Once)

```go
func main() {
	data, err := os.ReadFile("logs.txt")

	if err != nil {
		fmt.Println("Error reading file")
		return
	}

	fmt.Println(string(data))
}
```

## Manual Chunk Reading (No Buffer)

```go
file, _ := os.Open("file.txt")
defer file.Close()

buf := make([]byte, 1024)

for {
    n, err := file.Read(buf)
    if n > 0 {
        fmt.Print(string(buf[:n]))
    }
    if err != nil {
        break
    }
}

```

## Using bufio.Reader (Buffered Reading)

```go
file, _ := os.Open("file.txt")
defer file.Close()

reader := bufio.NewReader(file)

for {
    line, err := reader.ReadString('\n')
    if err != nil {
        break
    }
    fmt.Print(line)
}

```

- What if there is no new line?


## Scanner

```go
func main() {
	file, err := os.Open("logs.txt")

	if err != nil {
		fmt.Println("Error reading file")
		return
	}

	defer file.Close()

	scanner := bufio.NewScanner(file)

	for scanner.Scan() {
		line := scanner.Text()
		fmt.Println("========>", line)
	}

	err = scanner.Err()

	if err != nil {
		fmt.Println("Error reading the file")
		return
	}

}
```
