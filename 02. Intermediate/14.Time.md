# Time


## time.now()

- Current Local time.
- The system has a clock. Whatever time and date are logged into it, return that.

```go
func main() {
	t := time.Now()
	fmt.Println(t) // 2026-02-02 05:36:15.300169 +0530 IST m=+0.000224334
}
```


## time.Date()

- Creates a specific date value manually from the components you provide.

```go
func main() {
	t := time.Date(2026, time.January, 01, 6, 30, 0, 0, time.UTC)

	fmt.Println(t) // 2026-01-01 06:30:00 +0000 UTC
}
```
