# Time


## time.now()

- Current Local time.
- The system has a clock. Whatever time and date are logged into it, return that.

```go
func main() {
	t := time.Now()
	fmt.Println(t) // 2026-02-02 05:36:15.300169 +0530 IST m=+0.000224334
}
```


## time.Date()

- Creates a specific date value manually from the components you provide.

```go
func main() {
	t := time.Date(2026, time.January, 01, 6, 30, 0, 0, time.UTC)

	fmt.Println(t) // 2026-01-01 06:30:00 +0000 UTC
}
```

## time.parse(Layout, Input)

- Converts a string → `time.Time`, using a layout that describes how the input string looks.
- If the string doesn’t match the layout exactly, parsing fails.
- Go does NOT use tokens like YYYY-MM-DD. Instead, it uses this exact reference time `Mon Jan 2 15:04:05 MST 2006`.
- assumes UTC if no timezone is present.

| Input                  | Layout                 |
| ---------------------- | ---------------------- |
| `2026-02-02`           | `2006-01-02`           |
| `02/02/2026`           | `02/01/2006`           |
| `02 Feb 2026`          | `02 Jan 2006`          |
| `2026-02-02 14:30`     | `2006-01-02 15:04`     |
| `2026-02-02T14:30:00Z` | `2006-01-02T15:04:05Z` |
