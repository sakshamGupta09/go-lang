# String functions

- Strings are created using double quotes or backticks.
- Strings are immutable. Once created we cannot add, remove or replace characters.

## len(str)

- It counts the number of bytes in a string.
- English characters usually take 1 byte, so for english words len gives number of characters in a string.

```go
func main() {
	str := "Hello World"
	strJapenese := "こんにちは世界"

	fmt.Println(len(str))         // 11
	fmt.Println(len(strJapenese)) // 21
}
```

## concatenation

```go
func main() {
	firstName := "John"
	lastName := "Doe"

	fullName := firstName + lastName

	fmt.Println(fullName) // JohnDoe
}
```

## Strings are arrays

- Strings are array of characters.
- Characters are stored in memory as ASCII value.
- So, we can access character at any index using string[index]. Index starts from 0 to len-1.
- We can use array functionalities like slice.

```go
func main() {
	str := "Hello World"

	fmt.Println(str[0]) // 72
}
```

```go
func main() {
	str := "Hello World"

	fmt.Println(str[0:5])
}
```

## strconv.Itoa(int) string

- To convert a integer to a string. Because in go, string cannot be concatenated with numbers.

```go
func main() {
	str := "Hello World"

	num := strconv.Itoa(40)

	fmt.Println(len(num), str) // 2
}
```

## strings.Split(string, seperator) Array

- Converts a string into an array.

```go
func main() {
	fruits := "apple,banana,orange,litchi"

	parts := strings.Split(fruits, "-")

	fmt.Println(parts) // [apple,banana,orange,litchi]
}
```

## strings.Join(array, seperator) string

- Converts an array into a string.

```go
func main() {
	fruits := []string{"apple", "banana", "grapes"}

	parts := strings.Join(fruits, ", ")

	fmt.Println(parts) // apple, banana, grapes
}
```


## strings.Contains(string, substr) bool

- Checks if a substring is present in a string or not.

```go
func main() {
	str := "Hello World !"

	fmt.Println(strings.Contains(str, "ell")) // true
}
```

## strings.Replace(str, pattern, with, howManyOccourences)

```go
func main() {
	str := "Hello World !"

	fmt.Println(strings.Replace(str, "l", "KK", 3)) // HeKKKKo WorKKd !
}
```

## strings.TrimSpace(str)

- Removes trailing and leading spaces from a string.

```go
func main() {
	str := "    Hello World !   "

	fmt.Println(strings.TrimSpace(str)) // Hello World !
}
```

## strings.Repeat(str, n) string

```go
func main() {
	fmt.Println(strings.Repeat("*", 3)) // ***
}
```

## strings.Count(str, substr) int

- Counts the number of times a pattern is appearing in a string.

```go
func main() {
	str := "Hello World"
	fmt.Println(strings.Count(str, "l")) // 3
}
```

##  strings.hasPrefix/Suffix(string, pattern) bool

```go
func main() {
	str := "Hello World"
	fmt.Println(strings.HasPrefix(str, "He")) // true
	fmt.Println(strings.HasSuffix(str, "ld")) // true

}
```
