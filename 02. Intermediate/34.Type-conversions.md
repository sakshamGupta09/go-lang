# Type conversions

- Convert a value from one type to another related type.
- We cannot convert from one type to any type. There are some rules and restrictions.
- We use the type followed by parenthesis. Type(value)

## Related types

- Integer can be converted to `int8`, `int16`, `int32`, `int64`, `float32`, `float64`.
- A float can be converted to integer type.

```go
func main() {
	var num int = 32

	fmt.Println(int8(num))
	fmt.Println(int16(num))
	fmt.Println(int32(num))
	fmt.Println(int64(num))
	fmt.Println(float32(num))
	fmt.Println(float64(num))
}
```
```go
func main() {
	var num float32 = 32.18

	fmt.Println(int(num)) // 32
}
```
