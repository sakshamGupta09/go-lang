# Number Parsing

- To convert a textual representation of a number to a number.
- If the conversion is not possible, it throws an error.


## strconv.Atoi(string) int

- Converts a string to an `int` type.
- Works with decimal strings only. For hex it throws an error.

``` go
func main() {
	str := "12345"
	num, err := strconv.Atoi(str)

	if err != nil {
		fmt.Println("CONVERSION FAILED", err)
		return
	}

	fmt.Println("CONVERSION SUCCESSFUL", num)
}
```

## strconv.ParseInt(string, base, type)

- `Atoi` method always returns a `int` type. What if we want more granular tytpes like `int8`, `int16` etc.

```go
func main() {
	strHex := "FF"
	num, err := strconv.ParseInt(strHex, 16, 16)

	if err != nil {
		fmt.Println("CONVERSION FAILED", err)
		return
	}

	fmt.Println("CONVERSION SUCCESSFUL", num) // 255

	strBinary := "010101"
	num, err = strconv.ParseInt(strBinary, 2, 8)

	if err != nil {
		fmt.Println("CONVERSION FAILED", err)
		return
	}

	fmt.Println("CONVERSION SUCCESSFUL", num) // 21
}
```
