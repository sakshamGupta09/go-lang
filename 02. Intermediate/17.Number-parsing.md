# Number Parsing

- To convert a textual representation of a number to a number.
- If the conversion is not possible, it throws an error.


## strconv.Atoi(string) int

- Converts a string to an `int` type.
- Works with decimal strings only.

``` go
func main() {
	str := "12345"
	num, err := strconv.Atoi(str)

	if err != nil {
		fmt.Println("CONVERSION FAILED", err)
		return
	}

	fmt.Println("CONVERSION SUCCESSFUL", num)
}
```
