# IO Package

- I/O operations may involve reading data from multiple types of sources and writing data to multiple types of destinations. Now, if we keep defining different methods and types for each source, it will get complicated.
- `io` package helps to define basic io abstractions that will work with any data source and destinations. Because at the end, we just have to read data from a source and write data to a destination.
- It doesn’t care where data comes from or goes to — as long as it satisfies the interface.
- Sources: File, String, Network, Buffer
- Destinations: File, Stdout, Memory, Network


## Core Idea: Everything is a Reader or Writer


### io.Reader

- Reads data from a source into a byte slice. The data source can be anything file, stdin etc.

```go
type Reader interface {
    Read(p []byte) (n int, err error)
}
```

## io.Writer

-  Writes data from a byte slice to a destination.

```go
type Writer interface {
    Write(p []byte) (n int, err error)
}
```

## Common Implementations

- Let’s see real-world types that implement io.Reader / io.Writer.

## ✅ 1. File (os.File)

- `*os.File` implements both Reader and Writer. It is of type `io.Reader` and `io.Writer`.

```go
file, _ := os.Open("test.txt")
defer file.Close()

data := make([]byte, 100)
n, _ := file.Read(data)

fmt.Println(string(data[:n]))
```

## ✅ 2. bytes.Buffer

- An in-memory buffer. Implements `io.Reader` and `io.Writer`.

```go
var buf bytes.Buffer

buf.Write([]byte("Hello"))
buf.WriteString(" World")

fmt.Println(buf.String())
```

```go
data := make([]byte, 5)
buf.Read(data)
fmt.Println(string(data)) // Hello
```

- To build strings efficiently.


## ✅ 3. strings.Reader

```go
r := strings.NewReader("Hello Go")

data := make([]byte, 5)
r.Read(data)

fmt.Println(string(data)) // Hello
```

## ✅ 4. io.Copy — Power of Abstraction

```go
src := strings.NewReader("Hello")
dst := &bytes.Buffer{}

io.Copy(dst, src)

fmt.Println(dst.String()) // Hello
```
Here: src → string, dst → memory buffer. No special handling needed. Now change dst to a file

```go
file, _ := os.Create("out.txt")
defer file.Close()

io.Copy(file, strings.NewReader("Hello File"))
```


## io.Closer

- When reading form any resource, It's important to close that resource after we are done with it.

```go
func closeresource(c io.Closer) {
	err := c.Close()

	if err != nil {
		log.Fatal("Error while reading data", err)
	}
}
```
