# IO Package

- I/O operations may involve reading data from multiple types of sources and writing data to multiple types of destinations.
- io package helps to define basic io abstractions that will work with any data source and destinations.

## io.Reader interface

- Reads data from a source into a byte slice. The data source can be anything file, stdin etc

```go
func readFromSource(r io.Reader) {
	data := make([]byte, 1024)

	n, err := r.Read(data)

	if err != nil {
		log.Fatal("Error while reading data", err)
	}

	fmt.Println(string(data[:n]))
}
```

## io.Writer interface

```go
func writeToWriter(w io.Writer, data string) {
	n, err := w.Write([]byte(data))

	if err != nil {
		log.Fatal("Error while reading data", err)
	}

	fmt.Println("READ these many bytes", n)
}
```

## io.Closer

- When reading form any resource, It's important to close that resource after we are done with it.

```go
func closeresource(c io.Closer) {
	err := c.Close()

	if err != nil {
		log.Fatal("Error while reading data", err)
	}
}
```
