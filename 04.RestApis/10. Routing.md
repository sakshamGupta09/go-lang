# Routing

- Inside each route handler, we listen to the `r.Method` and handle requests.

## Http Methods

| Method      | Purpose (one sentence)                                                      |
| ----------- | --------------------------------------------------------------------------- |
| **GET**     | Retrieve data from the server without modifying anything.                   |
| **POST**    | Create a new resource or trigger an action on the server.                   |
| **PUT**     | Replace an existing resource with a new version.                            |
| **PATCH**   | Partially update specific fields of an existing resource.                   |
| **DELETE**  | Remove a resource from the server.                                          |
