# HTTP Versions


## HTTP

- Consider a client server architecture. Client sends a request to the server and server responds.
- This communication happens using a protocol called HTTP.
- A protocol is simply a set of rules for communication.
- First a TCP connection is established between client and server and then request response follows.


## HTTP/1.1

- I want to cummunicate with you several times. I have a lot of requests to make.
- Okay, for every request create a new connection with me.
