# Enconding

- Serialization is the process of converting GO objects to a transferable format like JSON or XML. We have two approaches to achieve this. Marshalling and Encoding.
- Marshalling and Unmarshalling works with bytes in memory. JSON is created in the memory first. Similatrly, JSON is read from the memory.


```go
Marshal:
Go Struct
    ↓
[]byte in memory
    ↓
Write somewhere

Encoder:
Go Struct
    ↓
Directly write JSON to destination
```

```go
Incoming HTTP Request
        │
        ▼
      r.Body
   (io.Reader)
        │
        ▼
json.NewDecoder
        │
        ▼
      Struct


      Struct
        │
        ▼
json.NewEncoder
        │
        ▼
        w
  (io.Writer)
        │
        ▼
Outgoing HTTP Response
```
