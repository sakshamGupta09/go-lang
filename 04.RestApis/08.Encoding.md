# Enconding

- Serialization is the process of converting GO objects to a transferable format like JSON or XML. We have two approaches to achieve this. Marshalling and Encoding.
- Marshal: JSON is created in memory. Then we print it somewhere.
- Instead of creating JSON in memory, write it directly to the destination like a file.
- w (http.ResponseWriter) is an io.Writer. The encoder directly writes JSON into the HTTP response.
- r.Body is an io.Reader. The decoder directly reads JSON from the request body.


```go
Marshal:
Go Struct
    ↓
[]byte in memory
    ↓
Write somewhere

Encoder:
Go Struct
    ↓
Directly write JSON to destination
```

```go
Incoming HTTP Request
        │
        ▼
      r.Body
   (io.Reader)
        │
        ▼
json.NewDecoder
        │
        ▼
      Struct


      Struct
        │
        ▼
json.NewEncoder
        │
        ▼
        w
  (io.Writer)
        │
        ▼
Outgoing HTTP Response
```
