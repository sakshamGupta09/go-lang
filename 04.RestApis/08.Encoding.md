# Enconding

- Serialization is the process of converting GO onjects to a transferable format like JSON or XML.
- `json.Marshal` / `json.Unmarshal` work with bytes in memory.
- `json.NewEncoder` / `json.NewDecoder` work with streams (io.Writer and io.Reader).


```go
Marshal:
Go Struct
    ↓
[]byte in memory
    ↓
Write somewhere

Encoder:
Go Struct
    ↓
Directly write JSON to destination
```
