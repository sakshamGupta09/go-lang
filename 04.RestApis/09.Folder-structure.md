## Folders and their purpose

```go
| Folder/File          | Purpose                    | What Goes Here                                                      | Example                            |
| -------------------- | -------------------------- | ------------------------------------------------------------------- | ---------------------------------- |
| `cmd/`               | Application entry points   | Executables that start your services                                | API server, worker, scheduler      |
| `cmd/server/main.go` | Bootstraps the application | Dependency wiring, configuration loading, route registration        | Start HTTP server                  |
| `internal/`          | Core application code      | Business-specific code that should not be imported by other modules | User, Order, Product modules       |
| `pkg/`               | Reusable shared utilities  | Generic code reusable across projects                               | Logger, Validator, Cache           |
| `configs/`           | Configuration files        | Environment-specific configs                                        | `local.yaml`, `prod.yaml`          |
| `migrations/`        | Database schema changes    | SQL migration scripts                                               | Create tables, add columns         |
| `scripts/`           | Operational scripts        | Build, deploy, maintenance scripts                                  | `build.sh`, `deploy.sh`            |
| `docs/`              | Documentation              | API docs, architecture docs                                         | OpenAPI specs                      |
| `tests/`             | Integration/E2E tests      | Cross-module tests                                                  | API tests                          |
| `go.mod`             | Module definition          | Module name and dependencies                                        | `module github.com/acme/ecommerce` |
| `go.sum`             | Dependency checksums       | Security verification of dependencies                               | Auto-generated                     |
| `README.md`          | Project documentation      | Setup and usage instructions                                        | Installation guide                 |

```
