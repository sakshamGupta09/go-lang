# Add certificates to our server

- We have spawned our server and asked users to come with their requests.
- But while clients are trying to reach our servers, they are getting a warning that our server is not secure.
- So, we need to add certificates to our server.


## Listen for Https requests

```go
	err := http.ListenAndServeTLS(serverAddress, "cert.pem", "key.pem", nil)
```

## Know Client details

- We have added certificates to our server and made it secure. But what about the clients that are trying to communicate with us.
- So, when a request reaches our server, we want to know which HTTP version and TLS version both client and server agreed to allow communication.
- Both client and server negotiates to decide which protocol and TLS version to use.

```go
func logRequestDetails(r *http.Request) {
	protocol := r.Proto
	tls := r.TLS

	fmt.Println(protocol, tls.Version)
}
```
