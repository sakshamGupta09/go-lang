# Extract information from request

- We want to read the information a request is carrying.
- That information can be in query, route params, form data, body, headers etc.
- We want the information in GO object format.


## Form Data

- Client is sending request which has information in form data.
- `r.ParseForm` parses the form data and converts it into a GO map with key value pairs. Value is a slice as form data supports sending multiple values for a single key.
- Map can be accessed via `r.Form`

```go
err := r.ParseForm()
if err != nil {
  return
}
fmt.Println(r.Form)
```

## Body

- `r.Body` is a stream and hence we use `io.ReadAll` to read all information.
- We must close the body after we read form it.
- We can use marshalling or decoding to convert JSON to GO objects.
- Always keep struct properties public so marshalling or decoding can happen.
- We can even store the request body in a map or struct based on convinience.

```go
data, err := io.ReadAll(r.Body)

if err != nil {
  return
}
defer r.Body.Close()

fmt.Println(string(data))
```

```go
var user User
err := json.NewDecoder(r.Body).Decode(&user)

if err != nil {
  return
}
defer r.Body.Close()
```

## Headers

- To read headers we use `r.Header` which returns a map[string] []string


## Query params

- It returns us a map with string and string slice as values.
- `map[string][]string`

```go
queryParams := r.URL.Query()
fmt.Println(queryParams)
```
