# Mux

- A Mux is a router for HTTP requests. It examines the incoming request and forwards it to the correct handler.
- http.handleFunc also uses mux behind the scenes.

```go
package main

import (
    "fmt"
    "net/http"
)

func usersHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintln(w, "Users Page")
}

func ordersHandler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintln(w, "Orders Page")
}

func main() {
    mux := http.NewServeMux()

    mux.HandleFunc("/users", usersHandler)
    mux.HandleFunc("/orders", ordersHandler)

    http.ListenAndServe(":8080", mux)
}
```
