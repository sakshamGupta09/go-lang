# Http Server

- Create a server that can listen to incoming requests and return proper response to the client.
- Server will have an address so clients know how to reach out to us.
- We also need a handler that tells what to do when a request comes to this server.

```go
func main() {
	serverAddress := "127.0.0.1:3000"

	err := http.ListenAndServe(serverAddress, nil)

	if err != nil {
		log.Fatal(err)
	}
}
```

## Endpoints and handlers

- Clients can now reach our server but we need to define endpoints so we understand better what a client needs from us and we will define handlers to handle that request.
