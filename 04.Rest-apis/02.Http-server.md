# Http Server

- Create a server that can listen to incoming requests and return proper response to the client.
- Server will have an address so clients know how to reach out to us.
- We also need a handler that tells what to do when a request comes to this server.

```go
func main() {
	serverAddress := "127.0.0.1:3000"

	err := http.ListenAndServe(serverAddress, nil)

	if err != nil {
		log.Fatal(err)
	}
}
```


## Request Handler signature

- Handler function takes two arguments writer which is of type `http.ResponseWriter` and a reader which is of type `*http.Request`

```go
func(w http.ResponseWriter, r *http.Request) {}
```

## Defining Routes

- Our server is up and running. But to better understand the ask of our clients, we need to define routes and each route will have a handler to serve a request.

```go
func main() {
	http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
		fmt.Fprintln(w, "My first server")
	})

	serverAddress := "127.0.0.1:3000"

	err := http.ListenAndServe(serverAddress, nil)

	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("Server is up and running on port: 3000")

}
```
