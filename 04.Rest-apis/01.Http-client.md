# Http Client

- To make HTTP requests and read response.

```go
client := &http.Client{}
resp, err := client.Get("https://dummyjson.com/products/1")
```

- It returns an error only if we are unable to reach to the URL. It might be invalid or network connectivity issue. Not 200 status codes do not fall into error category.
- If everything goes well we get a `*http.Response`.
- `response.body` is a stream, we wait till we read all the data from the server.
