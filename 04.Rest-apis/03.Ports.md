# Ports

- A single computer can run many network services simultaneously:

```go
Computer (192.168.1.10)

Port 80   → Web Server
Port 5432 → PostgreSQL Database
Port 6379 → Redis
Port 22   → SSH Server
```
- All these services share the same IP address because they all are running on same comouter.

## Common Ports

| Port | Service      |
| ---- | ------------ |
| 80   | HTTP         |
| 443  | HTTPS        |
| 22   | SSH          |
| 25   | SMTP (email) |
| 3306 | MySQL        |
| 5432 | PostgreSQL   |
| 6379 | Redis        |
