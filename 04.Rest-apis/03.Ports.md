# Ports

- Ports are to communicate with the outside world.
- A single computer can run many network services simultaneously on specific ports. The external world can access these processes running on our system through a port.
```go
Computer (192.168.1.10)

Port 80   → Web Server
Port 5432 → PostgreSQL Database
Port 6379 → Redis
Port 22   → SSH Server
```
- All these services share the same IP address because they all are running on same comouter.
- So, when a client sends us a request, they need to send us which computer they want to reach to (I.P) and also which service on this computer they want to interact with(Port number).

## Common Ports

| Port | Service      |
| ---- | ------------ |
| 80   | HTTP         |
| 443  | HTTPS        |
| 22   | SSH          |
| 25   | SMTP (email) |
| 3306 | MySQL        |
| 5432 | PostgreSQL   |
| 6379 | Redis        |
