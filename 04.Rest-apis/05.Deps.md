# Deps

## What happens when we run go get command

- The package and it's dependencies gets downloaded from the internet.
- The deps do not get downloaded to the project folder like node_modules. Go downloads the to a global module cache(Go installation folder/pkg/mod) on your machine.
- The mod file is updated with package names and version.

## Benifits

Go uses a shared cache:

- If 10 projects use Gin, Go downloads it only once.
- All projects reuse the same cached copy.
- This saves disk space and makes builds faster.
