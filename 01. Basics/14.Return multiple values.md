# Returning multiple values

- Sometimes, we might need to return multiple values from a function.
- We would generally use composite types like maps, objects or arrays to return multiple values and later destructure them.
- Error handling. We can resturn a result and an error from a function. Perform some logic, if it goes well give back teh result else give back the error.


## Syntax

- We use brackets (returnType1, returnType2) to declare that this function has multiple return values.
- Multiple values are returned using return V1, V2, V3 ....

```go
func main() {
	q, r := divide(10, 3)
	fmt.Println(q, r)

}

func divide(x int, y int) (int, int) {
	quotient := x / y
	remainder := x % y
	return quotient, remainder
}
```

## Error handling 

- Function can do some logic.
- If it succeeds, it will return the result.
- If it fails it gives an `error` on why it failed.

```go
func main() {
	res, err := compare(1, 1)
	fmt.Println(res, err)

}

func compare(x int, y int) (string, error) {
	if x > y {
		return "X is greater than Y", nil
	}
	if x < y {
		return "x is smaller than y", nil
	}
	return "", errors.New("Something went wrong")
}

```
