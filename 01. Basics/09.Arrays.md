# Arrays

- To store multiple values.
- Fixed size that will not change during program execution. It cant grow or shrink. If we know beforehand the size then it's a apt data structure.
- Index starts with 0 and last index is n-1.

## Declaration

```go
func main() {
	var fruits [5]string
}
```
- If we do not initialize the array, the elements will take a default value.

## Initialization

```go
func main() {
	var nums [5]int8 = [5]int8{1, 2, 3, 4, 5}

	fmt.Println(nums)
}

func main() {
	nums := [5]int8{1, 2, 3, 4, 5}

	fmt.Println(nums)
}
```

## Arrays are passed as value

- The variables are actually holding array values and not the address of memory location.
- So, when we assign some array to a new variable, a copy is created and passed to the new variable.
- Both variables are holding different values. Updating one won't affect the other.

```go
func main() {
	var originalList [3]int8 = [3]int8{1, 2, 3}
	var copy = originalList

	copy[0] = 0

	fmt.Println("ORIGINAL ARRAY", originalList) // ORIGINAL ARRAY [1 2 3]
	fmt.Println("COPY ARRAY", copy) // COPY ARRAY [0 2 3]
}
```

## Iterating over arrays

### Traditional for loop

- Since index starts at 0 and ends at n-1, so will our loop.
  
```go
func main() {
	var list [3]int8 = [3]int8{1, 2, 3}

	for i := 0; i < len(list); i++ {
		fmt.Println(i, list[i])
	}
}
```

### Range

```go
func main() {
	var list [3]int8 = [3]int8{1, 2, 3}

	for index, value := range list {
		fmt.Println(index, value)
	}
}
```

## Comparing arrays

- Length should be equal.
- Each element from first array is compared with corresponding element from array 2.

```go
func main() {
	a := [3]int8{1, 2, 3}
	b := [3]int8{1, 2, 3}

	fmt.Println("Are arrays equal", a == b) // true
}
```
