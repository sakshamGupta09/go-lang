# Exit

- Terminates the program immediately and returns an exit status to the operating system.
- Status zero means success and non zero means a failiure.
- No cleanups or defer functions registered will be executed.

```go
func main() {
	defer fmt.Println("AFTER EXIT??")

	fmt.Println("START MAIN")
	os.Exit(0)
	fmt.Println("AFTER EXIT")
}
```
