# Variadic params and functions

- When we declare a function, we need to define how many parameters it will receive and then we call that function with those many parameters.
- I want to call a function with variable number of parameters(0 to n). The list of parameters will be consolidated into a slice.


```go
func sum(nums ...int) int {
	sum := 0
	for _, v := range nums {
		sum += v
	}
	return sum
}
```
- We can combine regular parameters with Variadic parameter but variadic parameter must always be the last parameter in a function.


## Destructure slice or array elements

```go
func main() {
	nums := []int{1, 2, 3, 4, 5}
	fmt.Println(sum(nums...))
}

func sum(nums ...int) int {
	sum := 0
	for _, v := range nums {
		sum += v
	}
	return sum
}
```
