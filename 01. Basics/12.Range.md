# Range

- It's a convinient function that lets us iterate over collections like arrays, slices, maps, channels, strings.
- It provides a uniform way of iterating over collections for which otherwise we would need different syntax for different collections.
- Range creates a copy of each element. So, modifying this copy will not impact the original element.

## Syntax

- `for i,v := range collection`

```go
func main() {
	var list = []int{1, 2, 3, 4, 5}

	for i, v := range list {
		v = v * 2
		fmt.Println(i, v)
	}

	fmt.Println(list) // [1 2 3 4 5]

}
```


## Iterating over strings

- The value returned is the uniocode value of character.

```go
func main() {
	message := "Hello World"

	for i, v := range message {

		fmt.Println(i, v)

		/*
		   0 72
		   1 101
		   2 108
		   3 108
		   4 111
		   5 32
		   6 87
		   7 111
		   8 114
		   9 108
		   10 100
		*/
	}
}
```

## Blank identifier

- Range returns two values index and the element itself.
- But if we do not need index or element, we can use `_` identifier.

```go
func main() {
	var list = []int{1, 2, 3, 4, 5}

	for _, v := range list {
		fmt.Println(v)
	}
}
```
