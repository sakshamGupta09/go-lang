# Recover from panic

- After a `panic`, our program terminates. That means if during our program execution, a runtime panic happens, our program will terminate.
- `recover` lets us handle the panic gracefully, log it if we want and the program execution continues.


## Syntax

- recover function returns the panic value if a panic happened.
- If there is no panic, then the recover function returns `nil`.
- Always inside `defer`.

```go
func recovery() {
	if r := recover(); r != nil {
	}
}
```

```go
func main() {
	nums := []int{1, 2, 3, 4, 5}
	index := 10
	task(nums, index)
	fmt.Println("AFTER TASK IS COMPLETED")
}

func task(nums []int, index int) {
	defer func() {
		fmt.Println("INSIDE DEFER BLOCK")
		if r := recover(); r != nil {
			fmt.Println(r, "RECOVERY PHASE")
		}
	}()
	fmt.Println("START")
	value := nums[index]
	fmt.Println("FINISHED", value, index)
}
```
