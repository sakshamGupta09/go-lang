# Conditionals

## Simple if statement

```go
func main() {
	var age int = 25
	if age > 18 {
		fmt.Println("You are an adult")
	}
}
```

## If else

```go
func main() {
	var age int = 25
	if age > 18 {
		fmt.Println("You are an adult")
	} else {
		fmt.Println("You are not an adult")
	}
}

```

## If else if

```go
func main() {
	var age int = 59
	if age < 18 {
		fmt.Println("You are a kid")
	} else if age >= 18 && age < 50 {
		fmt.Println("You are an adult")
	} else {
		fmt.Println("You are getting older")
	}
}
```
