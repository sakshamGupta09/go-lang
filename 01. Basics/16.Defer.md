# Defer

- We have a function. From this function, we are calling another function. But that another function will be executed only when the first function completes.
- Postpones the execution of a function untill surrouding function returns or finishes execution.
- Do this cleanup at the end no matter how the function returns.


## Syntax

```go
func process(i int) {
	defer cleanup()
	fmt.Println("Normal execution", i)
}

func cleanup() {
	fmt.Println("CLEANUP FUNCTION")
}

// Normal execution 10
// CLEANUP FUNCTION
```
