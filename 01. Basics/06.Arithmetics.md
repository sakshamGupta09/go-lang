# Arithmetic

```go
package main

import "fmt"

func main() {
	var a, b int = 10, 20

	var result int = a + b
	fmt.Println("RESULTS:: ADDITION", result)

	result = a - b
	fmt.Println("RESULTS:: SUBTRACTION", result)

	result = a * b
	fmt.Println("RESULTS:: MULTIPLICATION", result)

	result = a / b
	fmt.Println("RESULTS:: DIVISION", result)

	result = a % b
	fmt.Println("RESULTS:: MODULAR DIVISION", result)
}

```

## Integer division

- When we perform division of two integer types, we always get the quotient which will be an integer.
- To get fractions, we need to ensure either one of the operands is a floating number.

```go
func main() {
	var a, b int = 10, 3
	var result int

	result = a / b
	fmt.Println("RESULTS:: DIVISION", result) // 3
}
```


```go
func main() {
	var result float64

	result = 10 / 3
	fmt.Println("RESULTS:: DIVISION", result) // 3
}
```
- Even though type of result is float, it still gives integer because we are dividing two integers.

```go
func main() {
	var result float64

	result = 10 / 3.0
	fmt.Println("RESULTS:: DIVISION", result) //  3.3333333333333335
}

```
