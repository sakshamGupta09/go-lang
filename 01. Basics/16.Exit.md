# Exit

- Terminate the program right now. No defer or cleanups should run.
- Status code is sent to the OS. 0 means program completed successfully, non zero means failiure.

```go
func main() {
	defer fmt.Println("AFTER EXIT??")

	fmt.Println("START MAIN")
	os.Exit(0)
	fmt.Println("AFTER EXIT")
}
```

## Exit on fatal startup error

```go
if os.Getenv("PORT") == "" {
    fmt.Println("PORT not set")
    os.Exit(1)
}
```
