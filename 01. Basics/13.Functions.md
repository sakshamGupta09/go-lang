# Functions

- To create modular and reusable blocks of codes that does exactly one thing based on single responsibility principle.

## Syntax

- func keyword.
- name of the function. If it's a public function, it should be capitalCased. Private ones are lowercased.
- List of parameters with their types (p int, q string).
- Return type/s.

```go
func greetUser(name string) string {
  fmt.Println("WELCOME", name)
}
```
