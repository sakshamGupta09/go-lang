# Constants

- A value that is not gonna change during program execution.
- Constant variables are declared with `const` keyword.
- Constants must be initialized.
- Constants can be typed as well as non typed or infered.

```go
const AGE = 18;
```

## Multiple constants

- Related constants can be grouped into a single block.

```go
const (
  name = "saksham"
  age int  = 14
)
```
