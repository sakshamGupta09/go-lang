# Init function

- `Init` function can be used in any package to do some setup or initialization work for that package.
- Go runtime automatically calls this function before main.
- No parameters and no return value.
- Executed only once per package even if the package is imported multiple times.
- There can be multiple init functions in one package. They are executed squentially in order they are written.


```go
func init() {
	fmt.Println("INITIALIZER 1")
}

func init() {
	fmt.Println("INITIALIZER 2")
}

func main() {
	fmt.Println(" MAIN")
}
```
