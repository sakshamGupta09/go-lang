# Non blocking operations


## Non blocking receive operation

- Default case in select allows us to read values in a non blocking way.
- If none of the channels have emitted any value, default case will immediately run.

```go
func main() {
	ch1 := make(chan int)

	go func() {
		time.Sleep(time.Second)
		ch1 <- 1
	}()

	select {
	case data := <-ch1:
		fmt.Println("DATA received", data)
	default:
		fmt.Println("No channels are emitting yet")
	}

	fmt.Println("CODE COMPLETED IN MAIN")
}
```

## Non blocking send operation

```go
func main() {
	ch := make(chan int)

	select {
	case ch <- 1:
		fmt.Println("Sent value to the channel")
	default:
		fmt.Println("No receiver is ready")
	}

}
```

## Real time data

```go
func main() {
	data := make(chan int)
	quit := make(chan bool)

	go func() {
		for {
			select {
			case value := <-data:
				fmt.Println("Value received form data channel", value)
			case <-quit:
				fmt.Println("Stopping execution now")
				return
			default:
				time.Sleep(time.Millisecond * 500)
				fmt.Println("Waiting for a channel to send data")
			}
		}
	}()

	time.Sleep(time.Second * 2)
	quit <- true
}

```
