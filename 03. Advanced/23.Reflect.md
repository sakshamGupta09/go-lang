# Reflect

- For every variable we create, we need to define it's type.
- We know beforehand what kinf=d of value we want to store in this variable.

```go
var x int = 10
fmt.Println(x + 5)
```

- Compiler knows x is of type `int`. What all operations can be done. What fields and methods exists on it.
- But sometimes you don’t know the type beforehand.

## Why Is This Useful?

- Suppose you write a function that can accept anything.

```go
func printType(x any) {
	fmt.Println(reflect.TypeOf(x))
}
printType(10)
printType("hello")
printType(true)
```


## Example 1 — Inspecting a Type

```go
package main

import (
	"fmt"
	"reflect"
)

func main() {
	x := 42

	t := reflect.TypeOf(x)
	v := reflect.ValueOf(x)

	fmt.Println(t)
	fmt.Println(v)
}
```

## Inspecting Struct Fields

- Let's say we want to iterate over a `struct` type.
- We do not know the type of `struct` beforehand. Still we can iterate over it and get field value pairs.

```go
package main

import (
	"fmt"
	"reflect"
)

type User struct {
	Name string
	Age  int
}

func main() {
	u := User{
		Name: "Saksham",
		Age:  28,
	}

	t := reflect.TypeOf(u)
	v := reflect.ValueOf(u)

	for i := 0; i < t.NumField(); i++ {
		field := t.Field(i)
		value := v.Field(i)

		fmt.Println(field.Name, value)
	}
}
```
