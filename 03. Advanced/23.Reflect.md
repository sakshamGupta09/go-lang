# Reflect

- When we want to deal with some data, most of the times we know beforehand the type of this data.
- But sometimes you don’t know the type beforehand.
- So, we can assign them `any` type and ask Go to inspect the type while the program is running.

## The reflect Package

- While my program is running, Please inspect and tell me the type of this variable and it's value.

| Thing           	| Meaning              | Return Type 		|
| --------------- 	| -------------------- | -----------------  |
| `reflect.TypeOf`  | What type is this?   |  reflect.Type      |
| `reflect.Value` 	| Actual runtime value | reflect.Value      |


## Why Is This Useful?

- Suppose you write a generic function that can accept anything.

```go
func printType(x any) {
	v:= reflect.ValueOf(x);
	fmt.Println(x.Kind())
}
printType(10)
printType("hello")
printType(true)
```


## Example 1 — Inspecting a Type

```go
package main

import (
	"fmt"
	"reflect"
)

func main() {
	x := 42

	t := reflect.TypeOf(x)
	v := reflect.ValueOf(x)

	fmt.Println(t)
	fmt.Println(v)
}
```

## Utility methods on Reflect.Value

```go
func main() {
	x := 10

	v := reflect.ValueOf(x)
	t := v.Type()

	fmt.Println(v.IsNil())
	fmt.Println(v.IsZero())
	fmt.Println(t)
	fmt.Println(v.Kind() == reflect.Int)
}
```

## Inspecting Struct Fields

- Let's say we want to iterate over a `struct` type.
- We do not know the type of `struct` beforehand. Or let's say we are building a generic function that can take any struct and iterate over it.

```go
package main

import (
	"fmt"
	"reflect"
)

type User struct {
	Name string
	Age  int
}

func main() {
	u := User{
		Name: "Saksham",
		Age:  28,
	}

	t := reflect.TypeOf(u)
	v := reflect.ValueOf(u)

	for i := 0; i < t.NumField(); i++ {
		field := t.Field(i)
		value := v.Field(i)

		fmt.Println(field.Name, value)
	}
}
```

## Modifying Values

- Instead of passing variable, we pass the address of that variable to `reflect.valueOf` function since GO passes by value.

```go
package main

import (
	"fmt"
	"reflect"
)

func main() {
	x := 10

	v := reflect.ValueOf(&x)
	v = v.Elem()

	v.SetInt(20)

	fmt.Println(x)
}
```

## Modifying struct values

- It's important that `struct` fields are Capital case in order for reflection to work.
