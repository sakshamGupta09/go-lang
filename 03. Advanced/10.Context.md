# Context

- We have to perform a task. We can simply attach a context to our task which will give us some special abilities like:
- Cancel the ongoing task.
- Timeout or deadline.
- Provide some extra information (like request ID or auth information).


## Create a context

- We can create a root context `context.Context` or parent using two methods `context.TODO` and `context.Background`.
- `context.TODO` is used when we are unsure about which context to use or will replace it with a proper context in future. So, it's just a placeholder for now.
- All other methods like `withTimeout`, `withCancel` and `withDeadline` takes a parent context as parameter.


```go
func main() {
	rootCtx := context.Background()
}
```

## Cancellation

- On every context object we can listen to a signal `<- ctx.Done()`.
- Whenever we use cancel or timeout, that signal emits a value.

```go
func main() {
	rootContext := context.Background()

	ctx, cancel := context.WithCancel(rootContext)

	go func() {
		<-ctx.Done()
		fmt.Println("Stopped ...")
	}()

	cancel()
	time.Sleep(time.Second * 2)
}
```

## Timeout/ Deadline

- Instead of manually calling the `cancel`, start a timer and after some given time call the `cancel` automatically and send a signal.

```go
ctx, cancel := context.WithTimeout(context.Background(), 2*time.Second)
defer cancel()
```

## Values

```go
ctx := context.WithValue(context.Background(), "userID", 42)

id := ctx.Value("userID")
```


## Parent → Child relationship

```go
parent := context.Background()
child, cancel := context.WithCancel(parent)
```
```go
Request Context
   ├── DB Call
   ├── API Call
   └── Worker
```

- If parent is cancelled → child is also cancelled.
- Cancel parent, Everything cancels.


## Examples

❌ Without context (bad)

```go
func fetchData() {
    time.Sleep(5 * time.Second)
    fmt.Println("Done")
}
```
- There is no way to stop this task.

✅ With context

```go
func fetchData(ctx context.Context) {
    select {
    case <-time.After(5 * time.Second):
        fmt.Println("Done")
    case <-ctx.Done():
        fmt.Println("Cancelled:", ctx.Err())
    }
}
```

```go
ctx, cancel := context.WithTimeout(context.Background(), 2*time.Second)
defer cancel()

fetchData(ctx)
```
