# Context

- We have an ongoing operation or task.
- I want to cancel it, timeout it, provide a deadline and pass request scoped data.
- We define a function. Let it do what it is supposed to do and simply pass context as a parameter.

## Create a context

- We can create a root context or parent using two methods `context.TODO` and `context.Background`.
- `context.TODO` is used when we are unsure about which context to use or will replace it with a proper context in future. So, it's just a placeholder for now.
- All other methods like `withTimeout`, `withCancel` and `withDeadline` takes a parent context as parameter.


```go
func main() {
	rootCtx := context.Background()
}
```

## ctx.Done()

- If the context is cancelled or timedout, this channel will emit a value.
- This is a simple operation to check if a number is even or odd.
- We pass context to it and write the logic in the default case.


```go
func checkEven(ctx context.Context, num int) string {
	select {
	case <-ctx.Done():
		fmt.Println("Operation has been cancelled")
		return "Cancelled"
	default:
		if num%2 == 0 {
			return "This is an even number"
		}
		return "This is an odd number"
	}
}
```

- When the function is executed, the context was still alive and hence the default case ran.

```go
	rootCtx := context.Background()

	ctx, cancel := context.WithTimeout(rootCtx, time.Second * 2)
	defer cancel()

	value := checkEven(ctx, 10)
	fmt.Println(value)
```

- After one second the context will expire.
- Since we are waiting for two seconds, till then context already expired and when we executed the operation again, we get cancelled message.


```go
func main() {
	rootCtx := context.Background()

	ctx, cancel := context.WithTimeout(rootCtx, time.Second)
	defer cancel()

	value := checkEven(ctx, 10)
	fmt.Println(value)

	time.Sleep(time.Second * 2)

	value = checkEven(ctx, 10)
	fmt.Println(value)
}
```
- The problem with this operation is that we have to call it multiple times to check if the context has expired or not.

```go
func doTask(ctx context.Context) {
	for {
		select {
		case <-ctx.Done():
			fmt.Println("Operation cancelled")
			return
		default:
			fmt.Println("Working on the task ....")
		}
		time.Sleep(time.Millisecond * 500)
	}

}

func main() {
	rootCtx := context.Background()

	ctx, cancel := context.WithTimeout(rootCtx, time.Second*2)
	defer cancel()

	ctx = context.WithValue(ctx, "name", "John")

	go doTask(ctx)

	time.Sleep(time.Second * 3)

	/*
		Working on the task ....
		Working on the task ....
		Working on the task ....
		Working on the task ....
		Operation cancelled
	*/
}

```
