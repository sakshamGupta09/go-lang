# Goroutines

- It lets your program run multiple functions concurrently (at the same time).
- By default if we have to perform 3 tasks, the natural flow is complete Task 1 then do Task 2 and then Task3. With concurrency, we can do all 3 tasks at the same time.
- Many real-world tasks involve waiting like API calls, Database queries, File reading, Network requests. If we run them sequentially, the program wastes time waiting.
