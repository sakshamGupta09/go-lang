# Goroutines

- I want to perform 3 tasks. One way is to complete Task 1 first, then Task 2 and then Task 3. Here each task waits for the other one to complete.
- The better approach is tio perform all three tasks concurrently.

## Example

```go
package main

import (
	"fmt"
	"time"
)

func printNumbers() {
	for i := 1; i <= 5; i++ {
		fmt.Println(i)
		time.Sleep(time.Millisecond * 500)
	}
}

func main() {
	go printNumbers()

	fmt.Println("Main function")

	time.Sleep(time.Second * 3)
}
```
