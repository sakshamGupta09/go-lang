# sync.Once

- Multiple goroutines have a common piece of code.
- We want that piece of code to run only one time despite multiple goroutines trying to run it.


## Basic Example

```go
package main

import (
	"fmt"
	"sync"
)

var once sync.Once

func initialize() {
	fmt.Println("Initializing...")
}

func main() {
	for i := 0; i < 5; i++ {
		go func() {
			once.Do(initialize)
		}()
	}

	var input string
	fmt.Scanln(&input)
}
```
