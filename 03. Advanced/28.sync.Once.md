# sync.Once

- A piece of code runs exactly one time, no matter how many goroutines try to run it.


## Basic Example

```go
package main

import (
	"fmt"
	"sync"
)

var once sync.Once

func initialize() {
	fmt.Println("Initializing...")
}

func main() {
	for i := 0; i < 5; i++ {
		go func() {
			once.Do(initialize)
		}()
	}

	var input string
	fmt.Scanln(&input)
}
```
