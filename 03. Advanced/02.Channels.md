# Channels

- Goroutines run independently of each other. There is no communication or sync between goroutines.
- Channels let goroutines communicate with each other, synchronize themselves and exchange data.
- Communication will always happen between two goroutines using channels.


## Send data to a channel

```go
channel <- value
```


## Receive data from a  channel

```go
value, ok := <- channel
```
- Channel returns two values, the data and whether the channel is closed or not.

## Types

- Buffered
- Unbuffered
