# Channels

- To enable communication between goroutines.
- Suppose one goroutine produces data and the othe one consumes and then processes that data. How do we make sure data is available for goroutine2.
- A channel is a pipe for communication between goroutines. Using this pipe, multiple goroutines can send and receive data.
- Channels are typed, meaning they carry a specific type of data.


## Creating a Channel

```go
ch := make(chan int)
```

## Sending Data to a Channel

- Pipe is towards a channel. So, data is going inside a channel.

```go
ch <- 10
```

## Receiving Data from a Channel

```go
value := <-ch
```
