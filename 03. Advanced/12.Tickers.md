# Tickers

- Timer → runs once after some time.
- Timer → Run again and again at intervals.
- You use tickers when something needs to happen continuously at fixed intervals, like:
- Polling an API every 5 seconds, Sending heartbeat signals, Monitoring (CPU, memory, logs).
- Tickers send a signal every interval via a channel.

## Problem with time.Sleep() in loops

```go
for {
	time.Sleep(2 * time.Second)
	fmt.Println("Running task")
}
```
- Hard to stop gracefully.
- No control (pause/reset)
