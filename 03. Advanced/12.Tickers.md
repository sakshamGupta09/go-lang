# Tickers

- Timer → Execute this code once after T time.
- Timer → Execute this code again and again at every T interval.
- You use tickers when something needs to happen continuously at fixed intervals, like:
- Polling an API every 5 seconds, Sending heartbeat signals, Monitoring (CPU, memory, logs).
- Tickers send a signal every interval via a channel.

## Problem with time.Sleep() in loops

```go
for {
	time.Sleep(2 * time.Second)
	fmt.Println("Running task")
}
```
- Hard to stop gracefully.
- No control (pause/reset)


## Basic Ticker Example

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	ticker := time.NewTicker(2 * time.Second)

	for t := range ticker.C {
		fmt.Println("Tick at", t)
	}
}
```
- `time.NewTicker()` creates a ticker
- `ticker.C` is a channel
- Every 2 seconds → it sends current time into the channel
- `range` keeps receiving forever
- Range loop is blocking in nature
