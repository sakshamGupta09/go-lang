# Buffered Channels

```go
ch := make(chan int, 2)
```

- Has storage (capacity = 2) and can store values. So, sender and receiver dont need to be ready at the same time and block execution.
- Sender can send without waiting (until full).
- Receiver can receive without waiting (if not empty).
- I don't want immediate synchronisation. Sender can keep sending values that will be stored in a buffer and receiver can read from the buffer.
- When we read from the channel, first value is removed and slot becomes available.


| Feature         | Unbuffered                    | Buffered               |
| --------------- | ----------------------------- | ---------------------- |
| Storage         | ❌ None                        | ✅ Yes                 |
| Send blocks?    | Always (until receiver ready) | Only when buffer full  |
| Receive blocks? | Always (until sender ready)   | Only when buffer empty |
| Use case        | Synchronization               | Data passing / queues  |


## Non blocking 

```go
func main() {
	ch := make(chan int, 2)

	ch <- 10
	fmt.Println("Value sent")
}
```
- Since buffer is not full, send does not wait for anything and doesn't blocks execution.
- With unbuffered channels, we were not able to do this. Sender blocks and waits for the receiver.

```go
func main() {
	ch := make(chan int, 2)

	ch <- 1
	ch <- 2
	ch <- 3

	fmt.Println("Value sent")
}
```
- Now the sender will block and wait for the receiver because we have exceeded capacity and cannot hold the values anymore in the buffer.

```go
func main() {
	ch := make(chan int, 2)

	ch <- 1
	ch <- 2

	go func() {
		time.Sleep(2 * time.Second)
		fmt.Println("RECEIVED", <-ch)
	}()

	ch <- 3
	fmt.Println("Value sent")
}
```
- While sending the third value, sender will block as buffer is full.
- After 2 seconds, we get a receiver. So value one moves put of the buffer and 3 is written to it.


```go
func main() {
	ch := make(chan int, 2)
	<-ch
}
```
- Receiver blocks if the buffer is empty as there is nothing to read.

```go
func main() {
	ch := make(chan int, 2)

	go func() {
		time.Sleep(2 * time.Second)
		ch <- 1
	}()

	fmt.Println(<-ch)
}
```
