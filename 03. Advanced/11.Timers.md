# Timers

- This code should execute after certain amount of time not immediately.

## time.Sleep

- Blocks the goroutine completely.
- Can’t reset, stop or timeout it.

##  Basic Timer Example

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	timer := time.NewTimer(2 * time.Second)

	fmt.Println("Waiting...")

	<-timer.C  // waits for timer to fire

	fmt.Println("Timer fired!")
}
```

- `time.NewTimer()` creates a timer which will send a value into channel after 2 seconds. It is non blocking.
- `timer.C` is a channel.
- After 2 seconds → it sends a value into the channel
- `<-timer.C` waits for that signal.

## Timeout Pattern

```go
select {
case msg := <-dataChan:
	fmt.Println("Received:", msg)

case <-time.After(2 * time.Second):
	fmt.Println("Timeout! No data received")
}
```

```go
func longRunningTask() {
	for i := range 20 {
		fmt.Println(i)
		time.Sleep(time.Second * 1)
	}
}

func main() {
	timeout := time.After(time.Second * 2)
	done := make(chan bool)

	go func() {
		longRunningTask()
		done <- true
	}()

	select {
	case <-done:
		fmt.Println("Received value from done channel")
	case <-timeout:
		fmt.Println("Timeout executed")
	}
}
```

## Resetting a Timer

- Restarts an expired or stopped channel. 

```go
timer := time.NewTimer(2 * time.Second)

<-timer.C
fmt.Println("First done")

timer.Reset(2 * time.Second)

<-timer.C
fmt.Println("Second done")
```

## Stopping a Timer

```go
timer := time.NewTimer(2 * time.Second)

stopped := timer.Stop()

if stopped {
	fmt.Println("Timer stopped before firing")
}
```
