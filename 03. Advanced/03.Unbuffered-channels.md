# Unbuffered channel

- It is a channel with no storage. It cannot hold data.
- It only transfers data directly from sender to receiver.
- So, communication happens only when both are ready at the same time.
- If any of them is not ready, we wait and block execution till the other party is ready.

## Create a unbuffered channel

```go
ch := make(chan int) // unbuffered channel
```

## Basic Example

```go
func main() {
    ch := make(chan int)

    go func() {
        ch <- 10 // send
        fmt.Println("Sent")
    }()

    value := <-ch // receive
    fmt.Println("Received:", value)
}
```
