# Benchmarking

- The code I wrote is giving me the expected output but is it performant?
- To measure performance of our code. How much time it take for our function or operation to complete.
- Use this to analyse how future changes made to this function are impacting performance.


## How to benchmark

- We prefix the function we want to benchmark with `Benchmark` keyword.
- Go decides how many times it should run the function to give us some analytics about performance.
- We run the benchmarking file using `go test -benchmark=. filename `

```go
func BenchmarkAdd(b *testing.B) {
	for range b.N {
		Add(-1, 1)
	}
}
```
