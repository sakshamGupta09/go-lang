# Testing

- Testing is the process to verify that our code is working as expected. It does what we expect it to do.
- Ensure the firther changes to our codebase do not break any existing implementations.
- Documentation on how the code should function.

## Prerequisites

- File suffix must be `_test.go`.
- Command to run the test file is `go test filename`.
