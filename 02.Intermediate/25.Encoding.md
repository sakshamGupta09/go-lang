# Encoding

- Serialization is the process of converting GO objects to a transferable format like JSON or XML. We have two approaches to achieve this. Marshalling and Encoding.
- Marshalling and Unmarshalling works with bytes in memory. I will write JSON to the memory and read JSON from the memory.
- Encoding/Decoding works with streams(io.Writer and io.Reader). I will write JSON to a destination(io.Writer) and I can read JSON from a source(io.Reader).


```go
Marshal:
Go Struct
    ↓
[]byte in memory
    ↓
Write somewhere

Encoder:
Go Struct
    ↓
Directly write JSON to destination
```

## Sending JSON Response

```go
func GetUser(w http.ResponseWriter, r *http.Request) {
    user := User{
        Name: "John",
        Age:  25,
    }

    json.NewEncoder(w).Encode(user)
}
```
- w is a `io.Writer`
- The encoder directly writes JSON into the HTTP response.


## Reading JSON Request

```go
func CreateUser(w http.ResponseWriter, r *http.Request) {
    var user User

    err := json.NewDecoder(r.Body).Decode(&user)
    if err != nil {
        http.Error(w, err.Error(), http.StatusBadRequest)
        return
    }

    fmt.Println(user.Name)
}
```
- r.Body is an `io.Reader`.
- The decoder directly reads JSON from the request body.

```go
Incoming HTTP Request
        │
        ▼
      r.Body
   (io.Reader)
        │
        ▼
json.NewDecoder
        │
        ▼
      Struct


      Struct
        │
        ▼
json.NewEncoder
        │
        ▼
        w
  (io.Writer)
        │
        ▼
Outgoing HTTP Response
```
